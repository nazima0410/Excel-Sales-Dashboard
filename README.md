# Excel-Sales-Dashboard
As part of my learning and internship journey, I built an interactive dashboard to analyze sales data and generate meaningful business insights.

📊 Key Highlights of the Dashboard: 
✅ Total Sales KPI Visualization
✅ Sales Analysis by Category
✅ Sales Analysis by Region
✅ Year-wise Sales Trend Analysis
✅ Interactive Filters using Slicers

🛠 Tools Used:
• Microsoft Excel
• Pivot Tables
• Pivot Charts
• Conditional Formatting
• Data Visualization Techniques

This project helped me strengthen my skills in data analysis, dashboard designing, and storytelling through data.
